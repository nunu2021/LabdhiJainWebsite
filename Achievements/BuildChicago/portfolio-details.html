Build Chicago - Winning Project : Syrian Tutoring

  • 1st Place Winner
  • Language: Swift (iOS)
  • Project date: 22 November, 2019
  • Source Code: https://github.com/nunu2021/SyrianTutoring

Syrian Tutoring is an iOS app that used a drawing feature to allow college students from first and second world countries to tutor refugee children in countries like Syria and Palestine. The app submitted was worked on for only 15 hours, and therefore is the abridged representation of the project idea.
I worked on this project with Siddhi Jain.
